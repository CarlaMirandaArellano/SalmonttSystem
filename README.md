# SalmonttSystem
Sistema para salmonera Salmontt

Organización en Paquetes: model y app.

Encapsulamiento: Atributos private y acceso vía métodos public (get/set).

Clases Base y Heredada: Persona y Empleado.

Composición: Persona (y por ende Empleado) contiene un objeto Direccion.

Reutilización: Empleado reutiliza toda la lógica de Persona mediante herencia.
